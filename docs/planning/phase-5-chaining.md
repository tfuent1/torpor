# Phase 5 — Request Chaining & Advanced Features

## Goal
Enable complex multi-step workflows and close the remaining feature gap with Insomnia/Postman.

## Deliverables

- [ ] Extract block — pull values from responses into environment variables
- [ ] Pre/post request script hooks (environment variable manipulation only)
- [ ] Response history with timestamp and search
- [ ] Response diff view (compare two history entries)
- [ ] Code generation: curl, PHP Guzzle, Python requests, JavaScript fetch
- [ ] Auth helpers: OAuth 2.0 client credentials flow
- [ ] WebSocket support (stretch goal)

## Definition of Done

A developer can authenticate via a login endpoint, automatically capture the token, and use it in subsequent requests — all without manual copy-paste. Response history is searchable and comparable.
